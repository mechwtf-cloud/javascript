📌 About

This project is a collection of browser-based JavaScript games built using simple web technologies.

The goal is to create fun, interactive mini-games while learning and improving coding skills.

🎯 Features
  🕹️ Multiple playable games
  ⚡ Runs directly in the browser (no install needed)
  🎮 Keyboard-based controls
  💻 Lightweight and fast
🛠️ Technologies Used
  HTML
  CSS
  JavaScript
▶️ How to Run Locally
  git clone https://github.com/mechwtf-cloud/javascript.git
  cd javascript

  Then just open index.html in your browser.
